# ThreatLog Smart Contract

## Deployment Instructions

1. Open Remix IDE: https://remix.ethereum.org
2. Create ThreatLog.sol and paste the contract code
3. Compiler settings:
   - Solidity version: 0.8.0 or higher
   - EVM version: paris
4. Deploy & Run Transactions:
   - Environment: Dev - Ganache Provider
   - RPC URL: http://127.0.0.1:7545
5. Click Deploy
6. Copy the deployed contract address
7. Update CONTRACT_ADDRESS in contract_config.py
